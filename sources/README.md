# Sources

This directory stores source material and source manifests for the investment research wiki.

Use `sources/assets/` for local screenshots, PDFs, images, and attachments when they can be safely committed. Keep source summaries, OCR, text extractions, and other transformed `.txt` or `.md` files in the same directory as the source assets they describe. Use source manifest files in `sources/` when the original source lives outside the repository or should not be copied into git.

Do not edit source content after ingest unless explicitly directed. Prefer updating the synthesized wiki pages in `wiki/`.
